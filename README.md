# labtour

Labtour adalah sebuah aplikasi Rekomendasi Destinasi Wisata di Indonesia berdasarkan tingkat resiko penularan covid-19

Teknologi yang digunakan untuk membangun aplikasi ini antara lain:
- Flutter
- Laravel
- Firebase
- SQLITE
